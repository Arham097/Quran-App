### How to Run the Project

1. Clone the repository:
    ```bash
    https://github.com/Arham097/Quran-App.git
    ```

2. Install dependencies:
    ```bash
    npm install
    ```

3. Start the development server:
    ```bash
    npm run dev
    ```

4. Access the app at:
    ```bash
    http://localhost:5173
    ```
